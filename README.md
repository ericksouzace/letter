# Letreiro Pro PWA

Arquivos para publicar no GitHub Pages:

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

No GitHub, suba estes arquivos na raiz do repositório. Depois vá em Settings > Pages e selecione `main` e `/root`.

Importante: não suba apenas o ZIP no repositório. O GitHub Pages precisa enxergar os arquivos soltos.
